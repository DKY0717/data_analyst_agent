# v0.6 分层分析意图、Schema Grounding 与主动澄清 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建可解释、可分层评测的分析意图理解、Schema Grounding、图路由和主动澄清链路，并通过严格质量门禁保持原有结果正确性与安全能力不回退。

**Architecture:** Safety Intent Guard 保持唯一入口；新增混合 Analysis Intent Parser、Schema Grounder、Graph Router、Risk Calibrator、Clarification Manager 和 Context Builder。所有层使用稳定结构化契约，低风险请求进入裁剪 Schema SQL 生成，高风险请求返回主动澄清，中风险请求最多扩大一次候选后重新推理。

**Tech Stack:** Python 3.11、Pydantic 2、LangGraph、DuckDB、PyYAML、SQLGlot、pytest、Qwen API、GitHub Actions

```python
from typing import Any, Literal

from pydantic import BaseModel, Field


class IntentSlot(BaseModel):
    concept: str
    confidence: float = Field(ge=0, le=1)
    evidence: str = ""


class FilterSlot(IntentSlot):
    operator: str
    value: Any


class RankingSlot(BaseModel):
    direction: Literal["asc", "desc"]
    limit: int = Field(gt=0)


class AnalysisIntent(BaseModel):
    task_types: list[str] = Field(default_factory=list)
    metrics: list[IntentSlot] = Field(default_factory=list)
    dimensions: list[IntentSlot] = Field(default_factory=list)
    filters: list[FilterSlot] = Field(default_factory=list)
    time_granularity: str | None = None
    ranking: RankingSlot | None = None
    missing_slots: list[str] = Field(default_factory=list)
    conflicts: list[dict[str, Any]] = Field(default_factory=list)
    overall_confidence: float = Field(default=0, ge=0, le=1)


class GroundingCandidate(BaseModel):
    candidate_id: str
    concept: str
    expression: str
    tables: list[str]
    columns: list[str]
    score: float = Field(ge=0, le=1)
    evidence: list[str] = Field(default_factory=list)


class GroundingResult(BaseModel):
    concept: str
    candidates: list[GroundingCandidate] = Field(default_factory=list)


class SchemaRoute(BaseModel):
    selected_tables: list[str] = Field(default_factory=list)
    join_edges: list[tuple[str, str]] = Field(default_factory=list)
    evidence: dict[str, list[str]] = Field(default_factory=dict)
    confidence: float = Field(default=0, ge=0, le=1)


class ClarificationOption(BaseModel):
    candidate_id: str
    label: str
    description: str


class ClarificationRequest(BaseModel):
    clarification_id: str
    reason: str
    question: str
    options: list[ClarificationOption]
    max_rounds: int = Field(default=2, ge=1, le=2)
```
