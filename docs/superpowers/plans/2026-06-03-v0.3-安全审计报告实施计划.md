# v0.3 安全审计报告 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 让每次自然语言查询都返回结构化 `audit_report`，展示 SQL 生成、Guard 校验、LIMIT 注入、修复和执行过程中的安全证据。

**Architecture:** 新增轻量 `AuditReportBuilder` 统一创建事件和报告；SQL Guard 返回 `audit_events` 和 `limit_injected`；AgentGraph 在共享 state 中追加生成、校验、修复、执行事件，API 将最终报告返回给前端。

**Tech Stack:** FastAPI、LangGraph TypedDict state、Pydantic、SQLGlot、pytest。
