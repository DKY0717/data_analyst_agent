## SQL 与错误摘录

### monthly_sales_2024

- 问题：统计 2024 年每个月的销售额
- SQL：`SELECT STRFTIME(o.order_date, '%Y-%m') AS month, SUM(o.total_amount) AS sales_amount FROM orders AS o WHERE o.order_date >= '2024-01-01' AND o.order_date < '2025-01-01' GROUP BY STRFTIME(o.order_date, '%Y-%m') ORDER BY month LIMIT 1000`
- 错误：无

### top_products_by_sales

- 问题：找出销售额最高的 5 个商品
- SQL：`SELECT p.product_name, SUM(oi.quantity * oi.unit_price) AS sales_amount FROM orders AS o JOIN order_items AS oi ON o.order_id = oi.order_id JOIN products AS p ON oi.product_id = p.product_id GROUP BY p.product_name ORDER BY sales_amount DESC LIMIT 5`
- 错误：无

### customer_count_by_region

- 问题：统计各地区的客户数量
- SQL：`SELECT r.region_name, COUNT(DISTINCT c.customer_id) AS customer_count FROM customers AS c JOIN regions AS r ON c.region_id = r.region_id GROUP BY r.region_name ORDER BY customer_count DESC LIMIT 1000`
- 错误：无

### sales_by_region

- 问题：按地区统计 2024 年销售额
- SQL：`SELECT r.region_name, r.province, r.city, SUM(o.total_amount) AS 销售额 FROM orders AS o JOIN customers AS c ON o.customer_id = c.customer_id JOIN regions AS r ON c.region_id = r.region_id WHERE STRFTIME(o.order_date, '%Y') = '2024' GROUP BY r.region_name, r.province, r.city ORDER BY 销售额 DESC LIMIT 1000`
- 错误：无

### sales_by_category

- 问题：统计各商品类别的销售额
- SQL：`SELECT c.category_name, SUM(o.total_amount) AS sales_amount FROM orders AS o JOIN order_items AS oi ON o.order_id = oi.order_id JOIN products AS p ON oi.product_id = p.product_id JOIN categories AS c ON p.category_id = c.category_id GROUP BY c.category_name ORDER BY sales_amount DESC LIMIT 1000`
- 错误：无

### refund_rate_by_category

- 问题：分析各商品类别的退款率
- SQL：`SELECT c.category_name, COUNT(DISTINCT r.refund_id) * 1.0 / NULLIF(COUNT(DISTINCT o.order_id), 0) AS refund_rate FROM orders AS o JOIN order_items AS oi ON o.order_id = oi.order_id JOIN products AS p ON oi.product_id = p.product_id JOIN categories AS c ON p.category_id = c.category_id LEFT JOIN refunds AS r ON o.order_id = r.order_id GROUP BY c.category_name ORDER BY refund_rate DESC LIMIT 1000`
- 错误：无

### repeat_purchase_rate

- SQL：`WITH customer_order_stats AS (SELECT o.customer_id, COUNT(DISTINCT o.order_id) AS order_count FROM orders AS o GROUP BY o.customer_id) SELECT COUNT(DISTINCT CASE WHEN cos.order_count > 1 THEN cos.customer_id END) * 1.0 / COUNT(DISTINCT cos.customer_id) AS 复购率 FROM customer_order_stats AS cos LIMIT 1000`
